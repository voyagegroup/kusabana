kusabana
========

test proxy for Elasticsearch

### Require

memcached  
ruby => 2.1.1 (maybe work also in ruby => 2.0)

### Installation

    git clone https://github.com/voyagegroup/kusabana.git
    cd kusabana
    bundle install

More documents are available at http://voyagegroup.github.io/kusabana/
